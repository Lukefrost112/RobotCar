# PS5 Controlled Smart Car

ESP32 smart car controlled via PS5 DualSense.

## Highlights
- Differential drive mixing (throttle + steering)
- Non-blocking feedback (RGB + buzzer)
- Priority-based buzzer state machine (PATTERN > BOOST > HORN > REVERSE > STANDBY)
